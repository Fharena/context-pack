# Context Pack v0.2.18

Public beta release focused on agent-first UX, release evidence, and first-run routing quality.

## Highlights

- Added public beta dogfood benchmarks on real public repos.
- Added launch copy and a Codex skill audit.
- Routed natural CI/build failure prompts such as `CI is red` and `build failed` to automation/source/tests.
- Improved first-run source inference for Python repos with top-level package directories such as `httpx/`.
- Kept natural bug/review/handoff routing covered through packaged npm CLI smoke tests.

## Evidence

- 78 stdlib unit tests.
- Packaged npm CLI validation.
- Python wheel/sdist build and `twine check`.
- npm tarball dry run.
- GitHub Actions CI on Windows and Ubuntu with Python 3.11/3.12.

## Install

GitHub install path:

```bash
npx github:Fharena/context-pack install-codex --activate
```

Repo setup for Claude, Cursor, other agents, or direct terminal use:

```bash
npx github:Fharena/context-pack setup
```

Registry publishing is prepared but remains dependent on npm/PyPI authentication or trusted publishing configuration.
